# PinUp GBP Manager 사용 가이드

**PinUp GBP Manager**는 식당 점주 및 마케터를 위해 제작된 Google Business Profile(GBP) 관리 AI 솔루션입니다.
복잡한 영문 마케팅 콘텐츠 생성부터 상세한 현황 진단 및 액션 플랜 수립까지 원스톱으로 지원합니다.

---

## 🚀 시작하기 전에 (설정 가이드)

프로그램을 정상적으로 사용하려면 **Google Gemini API Key**가 필요합니다.

1. [Google AI Studio](https://aistudio.google.com/)에 접속하여 무료 API Key를 발급받습니다.
2. 발급받은 API Key를 앱 좌측 사이드바의 **`⚙️ Configuration > Google API Key`** 입력란에 붙여넣습니다.
3. 매번 입력하기 번거롭다면, 프로그램 폴더 내 `.env` 파일을 생성하거나 열어 다음과 같이 설정하세요:
   ```env
   GOOGLE_API_KEY=여기에_발급받은_키를_입력하세요
   ```

## 📝 메뉴 1: GBP 콘텐츠 생성기

한국어로 식당의 기본적인 정보만 입력하면, AI가 자동으로 GBP에 최적화된 **고품질 영문 마케팅 콘텐츠**와 그 **한국어 번역본**을 생성합니다.

### 📌 사용 방법
1. 사이드바에서 **`📝 GBP 콘텐츠 생성기`** 메뉴를 선택합니다.
2. 다음 네 가지 필수 항목을 입력합니다:
   - **식당 이름 (Restaurant Name)** (예: 맛돌이 곱창)
   - **위치/주소 (Location)** (예: 서울 마포구 연남동 123-4)
   - **주요 메뉴 (Main Menu)** (예: 소곱창구이, 막창, 볶음밥)
   - **매장 특색 (Features & Vibe)** (예: 레트로 감성, 20년 전통, 현지인 맛집)
3. **`🚀 Generate GBP Content`** 버튼을 클릭합니다.

### 💡 생성되는 콘텐츠 결과 (4종 세트)
- **Description (비즈니스 소개글)**: 750자 이내의 검색 엔진 최적화(SEO) 반영 영문 소개글
- **Menu (메뉴판)**: 보기 좋게 카테고리화된 영문 메뉴 리스트
- **Q&A (자주 묻는 질문)**: 구글 사용자 질문/답변 10세트
- **Google Posts (구글 포스팅 템플릿)**: 업데이트, 오퍼(할인/이벤트), 행사, 음식 포커스 등의 4가지 포스트 샘플

*모든 결과물은 영문(🇺🇸)과 함께 자연스러운 한국어(🇰🇷) 번역본이 탭으로 구분되어 제공되므로, 내용을 쉽게 검토하고 활용할 수 있습니다.*

## 📊 메뉴 2: GBP 진단 리포트

현재 식당의 GBP 세팅 상태를 6가지 카테고리로 상세히 진단하여, 가시성 점수와 개선 액션 플랜을 도출합니다. 진단 결과는 PDF 리포트로 다운로드할 수 있습니다.

### 📌 사용 방법
1. 사이드바에서 **`📊 GBP 진단 리포트`** 메뉴를 선택합니다.
2. **Section 1: 식당 기본 정보**를 입력합니다. (식당명(한국어), 주소, 음식 카테고리 필수)
3. **Section 2: GBP 현황 진단 체크리스트**에서 현재 식당의 상태에 맞는 항목을 모두 선택합니다.
   - 각 카테고리(프로필 기본 정보, 영문 콘텐츠, 사진/영상, 리뷰 관리, 속성 설정, 멀티채널)를 클릭하여 아코디언 메뉴를 열고, 라디오 버튼을 체크하세요.
4. **Section 3: 진단 결과 대시보드 (실시간)** 에서 진단 현황을 확인합니다.
   - **종합 점수 게이지 (100점 만점)**: 5단계 등급(위험, 미흡, 보통, 양호, 우수)으로 현재 상태 시각화
   - **카테고리별 레이더 차트**: 강점과 취약점을 직관적으로 비교 파악
   - **개선 우선순위 자동 추출**: "즉시 개선 필요(🔴)" 요소와 "추가 최적화 권장(🟡)" 요소를 실시간으로 제안
5. 모든 진단이 끝나면 맨 하단의 **`📈 Report Generation`** 버튼을 클릭합니다.
6. **Section 4: AI 기반 종합 리포트 및 액션 플랜**이 생성됩니다. 생성된 내용은 `📥 Download PDF Report` 버튼을 눌러 깔끔한 리포트로 저장할 수 있습니다.

## 💡 유의사항 및 문제 해결
- **오류 발생 시**: API Key가 올바르게 입력되었는지 확인하세요. `Google API Key`가 없거나 한도가 초과된 경우 오류가 발생할 수 있습니다.
- **한국어 폰트 관련**: 진단 리포트 PDF를 저장할 때 한글이 깨지거나 표시되지 않는다면, `GBP` 폴더 내에 `NanumGothic.ttf` 파일이 정상적으로 위치해 있는지 확인해 주세요.
- **의존성 설치**: 애플리케이션을 로컬 환경에서 실행하려면 터미널(가상환경)에서 `pip install -r requirements.txt` 명령어를 실행하여 필요한 패키지가 모두 설치되어야 합니다. 특히 대시보드 시각화를 위해 `plotly` 패키지가 필수적입니다.
